# CodePulse 🚀

CodePulse is a professional, offline-first developer tracking dashboard and LeetCode workspace visualizer for Android. Built entirely in modern Kotlin and Jetpack Compose utilizing Material Design 3, CodePulse allows programmers to track, search, visualize, and locally export their synchronized coding activity from LeetCode and GitHub.

Whether you're compiling daily progress statistics, managing algorithmic practice goals, or exploring automated repositories of LeetCode solutions, CodePulse acts as your local hub for developer analytics and code archiving.

---

## 🎨 Visual Identity & Design Principles

Designed with a high-contrast **Slate Cyber Theme** and robust ergonomics:
*   **Edge-to-Edge Navigation**: Perfect handling of safe visual area content with Material 3 scaffolds and `WindowInsets` configurations.
*   **Unified Spacing**: Strict adherence to a clean, spacious 8dp density grid, offering excellent visual rhythm and layout balance on handheld displays.
*   **Context-Aware Information Scanning**: Color-coded categorization fields, detailed metadata boxes, progress sliders, and high-visibility difficulty meters (Easy/Medium/Hard) allow users to evaluate stats instantly.

---

## ⚙️ Core Modules & Features

### 📊 1. Consolidated Analytics Dashboard
A single, powerful entry point mapping all statistics, historical trends, and practice metrics:
*   **LeetCode Analytics**: View total problems solved alongside a difficulty breakdown (Kotlin-green for Easy, Solar-orange for Medium, Crimson-red for Hard). Displays your exact global user ranking and a list of detailed recent submissions showing status, timestamp, runtime, and memory metrics.
*   **GitHub Insights**: Highlights repository catalogs, profile creation history, total star counts, follower aggregates, and public commit metrics.
*   **Daily Analytics Log**: Automatically writes logs documenting your daily performance and progress in a clean timeline layout.

---

### 🎯 2. Insights, Goals & Milestones
An interactive target planner to promote daily coding discipline:
*   **Goal Tracker**: Create customized numeric practice thresholds (e.g., *"Solve 5 Hard Stack Problems"*). Supports instant creation dialogs and deletion flags.
*   **Achievements & Badges**: Unlock localized milestone badges (e.g., *"Database Architect"*, *"Consistent Coder"*, *"Algorithm Specialist"*) as you tick off milestones and goals.

---

### 📂 3. Solutions Repository Explorer (LeetCode Sync Explorer)
A comprehensive viewer dedicated to browsing programming solutions that are automatically pushed to GitHub repositories (such as those generated by automated browser tools like LeetHub):
*   **Authentication & Access**: Supports standard connecting configurations using Git coordinates (`owner/repository`) and Personal Access Tokens (PAT).
*   **Full Interactive Sandbox Mode**: No token? No problem. Instantly boots a pre-populated repository sandbox loaded with **50+ production-ready solutions** spanning 11 custom subdirectories (Dynamic Programming, Graph Theory, Trees, SQL Queries, Logic, etc.).
*   **Flexible Search Indexes**: Multi-criteria search filters allow finding code solutions instantly by problem ID, title phrase, topic, or target language.
*   **Code Presenter**: Styled with safe monospace syntax presentation, inline actions, customizable copying mechanisms, and direct reference hyperlinking back to the source repository.
*   **Bulk Workspaces & Topic Export**: Full ZIP generation framework. Bundle and export either individual solution scopes, complete categorizations (e.g., all *Trees* solutions), or your entire workspace repository compile directly into a `.zip` archive on your Android device downloads library.

---

### ⚙️ 4. Profile & Synchronization Control
Fine-tuned configurations to keep your workspace secure and fast:
*   **Data Synchronization**: Quick manual refresh buttons or automated background fetch configurations.
*   **Background Worker Framework**: Scheduled routines updating Room caches silently via Android's high-efficiency `WorkManager` API.
*   **Theme Switch**: Toggle between beautiful dark and light themes seamlessly.

## 🛠️ App Architecture & Stack

CodePulse follows Android's **Modern App Architecture (MAD)** guidelines based on clean, separated layer models:

```
├── UI Layer (Jetpack Compose, MVVM)
│   ├── ViewModels (Flow/StateFlow, state-retention, event loops)
│   └── Composable Screens (Polished Material 3 components)
│
├── Domain/Data Repository Unified Interface
│   └── CodePulseRepository (Data caching, search indexing, sync control)
│
└── Storage Layer (Data Sources)
    ├── Room SQLite Database (Highly optimized cache tables)
    ├── Encrypted / SharedPreferences (Credentials, tokens, states)
    └── Retrofit Web Interfaces (GitHub & LeetCode integrations)
```

### 📦 Key Dependencies
*   **Jetpack Compose**: Modern reactive declarative UI modeling.
*   **Material 3 (M3)**: Design token styling (color scheme maps, typography bindings).
*   **Room DB**: High-reliability offline SQLite mapping with KSP code processing.
*   **Retrofit & Serialization**: Type-safe REST communications and JSON parsers.
*   **WorkManager**: System-scheduled background data synchronizers.
*   **FileProvider**: Secure device file path sharing and ZIP generation.

## 🚀 Setup & Getting Started

### Prerequisites
*   **Android Studio Jellyfish+** (or latest preview)
*   **JDK 17+**
*   **Android SDK 34+**

### Local Run Configuration
1.  **Clone the project** repository to your developer workspace.
2.  **Open in Android Studio**, allowing the Gradle sync process to resolve and compile version indexes.
3.  **Run the application** on a connected physical device or emulator.
4.  Optionally, configure your **GitHub Profile & Credentials** securely via the **Settings/Profile** tab or dive straight into the features using the **Sandbox Playground** mode.
